Description

The module detects corner events in address events using the adapted Harris method.

Parameters

  • – name : Specifies the stem name of ports created by the module.
  • – strict : Sets ouput port to use strict protocol.
  • – width : Number of pixels on the x-axis of the sensor.
  • – height : Number of pixels on the y-axis of the sensor.
  • – filterSize : Size of Sobel filter used for the detection.
  • – tempsize : Length of the temporal window in seconds used to accumulate events.
  • – windowRad : Radius of the spatial window in pixels.
  • – sigma : Standard deviation of the Gaussian filter.
  • – thresh : Threshold for a confirmed corner event detection.
  • – nthreads : Number of threads used for the computation.

Input Ports

  • /vCorner/vBottle:i [eventdriven::vBottle] [default carrier:udp]: Accepts the address events in the vBottle container

Output Ports

  • /vCorner/vBottle:o [eventdriven::vBottle] [default carrier:udp]: Outputs corner events in the form of an eventdriven::LabelledAE. The vBottle also contains all events in the vBottle received as input.
  • /vCorner/score:o [yarp::os::Bottle] [default carrier:udp]: Outputs debug information for use with yarpscope. The gap between the time required to get and process events to detect data being lost.
